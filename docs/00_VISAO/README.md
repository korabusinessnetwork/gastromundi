# 00 — Visão

Visão de produto e identidade da GastroMundi. A **fonte de verdade** da identidade vive em `memory/` (raiz do repositório); esta pasta resume e dá contexto de produto.

## Resumo

GastroMundi é a plataforma all-in-one de gestão para **restaurantes e varejo**: PDV, Caixa, Pedidos, Cozinha, Estoque, Financeiro, Clientes e Relatórios, com o **Jarvas** (IA transversal) acompanhando todos os módulos para gerar insights, alertas e sugestões. A venda no PDV é a transação-fonte e se propaga automaticamente para os demais módulos.
